## Défi de la nuit 2025 :

Le but de ce défi était de créer un site parlant de la démarche NIRD, une association qui à pour but de distribuer des ordinateur pour les collèges et lycées en manque de matériel informatique en réutilisant des ordinateurs portables reconditionnés mis sous Linux.

Le sujet étant large, nous avons décider de faire un site de présentation avec un quiz de fin est des pages web correspondant aux autres défis.

## Hidden Snake :

Pour pouvoir accéder au Hiden Snake il suffit de trouver le "s" vert et souligné, caché dans la page d'acceuille !

Si jamais vous ne trouvez toujours pas, voici dans quel paragraphe il est caché :

"Les grandes idées derrière la démarche"

## La faille la plus maligne :

Une page d'administrateur ultra secrete et très sécurisé est malheureusement disponible sur le site depuis la barre de navigation à cause d'un manque d'organisation, si des gens malintentionné trouve le nom et mot de passe ils pourront accéder aux informations hyper sensible et top secret derriere le site web.

Un petit indice pour trouver la faille :
<details>
  tester n'importe quel combinaison de mot de passe et de nom d'utilisateur :)
</details>

## L'ergnomie : simplifier pour mieux vivre

Une page web avec une petite vidéo interessante concernant le projet est disponible depuis la barre de navigation mais etrangement cette page est très mal adapté au public, elle est tout simplement difficile est très enervante d'utilisation aux nombreuses erreurs d'ergonomie.
